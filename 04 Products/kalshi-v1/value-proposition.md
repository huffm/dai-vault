# value proposition: kalshi-v1

## core value
before every major economic release, know whether the Kalshi market price reflects what is actually knowable — without checking five sources yourself.

## what it replaces
- manually cross-referencing Kalshi prices against CMEFedWatch, BLS consensus, and economic calendars before each event
- gut-checking Polymarket or Manifold manually to see if another prediction market agrees
- trading on incomplete reference context because assembling it took too long

## why it is better than alternatives
| alternative | weakness | our difference |
|---|---|---|
| manual reference research | 20-30 minutes per event, often skipped | brief is assembled and scored 24 hours out |
| metaculus / manifold aggregates | no brief, no comparison to Kalshi prices | we surface the delta, not just the number |
| kalshi market page | shows price, not context | we compare price to every available reference |

## what makes it sticky
- accuracy track record: over time, users can see whether mispricing flags resolved correctly — this is a compounding trust signal
- event coverage: every major economic release is covered automatically, no manual setup
- the brief is the same format every time — users calibrate their trust in the scoring across events

## one-liner for acquisition
"see where the Kalshi crowd looks wrong before the number drops."
